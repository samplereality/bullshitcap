Limit meetings! Cap the bullshit! Take the pledge!

I propose the academic equivalent of the Norquist tax pledge: all new proposed new meetings must be offset by elimination of old meetings.

— Jimmy Lin (@lintool) May 1, 2015

I think we have way too meetings. As my colleague argues, we have financial budgets — why don't we have time budgets? In the same way that we need to balance our financial budgets (or otherwise make sure that our debt is sustainable), we need to balance our time budgets, since "time deficits" are not sustainable. I think there's a simple solution to this problem: if you want a new meeting, you have to eliminate an existing one.

What do you think? If you think this is a good idea: send a supportive tweet, clone this repo, modify this page to embed the tweet in it (order the tweets reverse chronologically earliest to latest), and send me a pull request!
